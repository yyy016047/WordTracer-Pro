# WordTracer Pro Edge浏览器测试指南

## 一、测试准备

1. 确保已安装最新版本的Microsoft Edge浏览器
2. 打开Edge浏览器，进入扩展管理页面：
   - 点击浏览器右上角的三点菜单
   - 选择「扩展」
   - 点击「管理扩展」
3. 开启「开发者模式」（页面右上角开关）

## 二、安装测试版插件

1. 在扩展管理页面，点击「加载解压缩的扩展」
2. 选择本项目的word-tracer文件夹
3. 插件应该会自动安装并出现在扩展列表中
4. 确保插件已启用

## 三、功能测试

### 1. 基础翻译功能
- 打开任意英文网页
- 选择一个英文单词
- 按快捷键「Q」（或自定义快捷键）
- 检查是否显示翻译弹窗
- 验证翻译内容是否正确

### 2. 单词历史记录
- 翻译多个不同的单词
- 点击插件图标，查看历史记录列表
- 检查历史记录是否按使用次数排序

### 3. 统计看板
- 点击插件图标，选择「统计看板」
- 查看统计数据是否正确显示
- 测试清空历史记录功能
- 测试设置功能（如调整每页显示数量、切换显示选项）

### 4. 快捷键设置
- 在统计看板页面，修改快捷键
- 返回网页测试新的快捷键是否生效

### 5. 全文高亮功能
- 翻译单词后，检查单词是否被高亮显示
- 测试取消高亮功能

## 四、兼容性测试

1. 测试在不同类型网页上的表现（新闻、博客、文档等）
2. 测试在不同Edge浏览器版本上的兼容性
3. 检查是否有控制台错误
