# WordTracer Pro

WordTracer Pro是一个浏览器扩展，专为CS专业人士和学生设计，提供高效的单词翻译、本地词库查询和学习管理功能，支持Chrome和Edge浏览器。

## 功能特性

### 1. 智能翻译
- 支持单词、短语和长句的实时翻译
- 集成有道、百度等多个翻译API
- 专业术语库支持，提供领域专用翻译

### 2. 本地词库
- 内置CS专业术语库（dict.json）
- 支持自定义专业翻译
- 本地词库单词显示专用标签

### 3. 音标显示
- 单个单词自动显示音标
- 支持英式和美式音标
- 本地词库单词也能正确显示音标

### 4. 学习管理
- 查询历史记录
- 单词高亮功能
- 学习看板管理

### 5. 用户体验优化
- 快捷键触发（默认Q键）
- 流畅的弹窗界面
- 支持长句换行显示

## 安装方法

1. 下载或克隆本项目
2. 打开浏览器（Chrome或Edge）
3. 进入扩展管理页面：
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
4. 开启右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"（Chrome）或"加载已解压的扩展"（Edge）
6. 选择项目文件夹

## 使用说明

1. **翻译单词/短语**：
   - 选择网页上的文本
   - 按 `Q` 键触发翻译
   - 查看翻译结果和音标

2. **使用学习看板**：
   - 点击浏览器扩展图标
   - 进入"学习看板"
   - 查看历史查询记录
   - 管理学习进度

3. **单词高亮**：
   - 在翻译弹窗中勾选"高亮"
   - 网页中所有相同单词将被高亮显示

## 配置说明

### 快捷键设置
- 网页内快捷键：默认`Q`，可在学习看板中自定义
- 全局快捷键：`Ctrl+Shift+Q`(Windows/Linux)或`Command+Shift+Q`(Mac)

### 专业术语库
- 词库文件：`dict.json`
- 支持添加自定义专业术语

## 技术栈

- HTML5 + CSS3
- JavaScript
- 浏览器扩展API
- 有道/百度翻译API
